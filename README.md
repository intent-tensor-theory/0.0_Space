This is a **college-level textbook**, not a manifesto and not a paper.  
Structure first, assumptions explicit, ascent disciplined.

Beginning at **Chapter 0.0** and ending *only when space is unavoidable*.

**No geometry is assumed until it is earned.**

This work has **resisted spatial reflexes all the way through.**  
Space was not smuggled in via:
- topology
- continuity
- "abstract manifolds"

More precisely: This is a **pre-theoretical compiler** for physics.  
It determines *where* other theories are allowed to begin.

- "space is fundamental" casually, or
- "background independence solves it" vaguely.
